# Thinkport API

Get data from Thinkport GmbH, Germany. It includes the following services:

* [x] Members
* [ ] About
  * [x] Locations
* [ ] Events
* [ ] News

## Usage

### Members

JavaScript with axios:

```javascript
const axios = require('axios');

axios.get('https://api.thinkport.andrelademann.de/members')
    .then((response) => {
        console.log(response.data);
    });
```
